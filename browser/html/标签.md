# html 5 标签
https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element
## 表单

```html
1. <button>	HTML <button> 元素表示一个可点击的按钮，可以用在表单或文档其它需要使用简单标准按钮的地方。
2. <datalist>HTML Datalist 元素 (<datalist>) 包含了一组<option>元素,这些元素表示其它表单控件可选值.
3. <fieldset>HTML form 元素 表示了文档中的一个区域，这个区域包含有交互控制元件，用来向web服务器提交信息。
4. <input>  HTML <input> 元素用于为基于Web的表单创建交互式控件，以便接受来自用户的数据。
5. <label>	HTML 元素表示用户界面中项目的标题。
6. <legend>	HTML的<legend>元素（也称为HTML的域说明元素（or HMTL
 Legend Field Element））代表一个用于表示它的父元素<fieldset>的内容的标题。</legend>
7. <meter>	HTML <meter>元素用来显示已知范围的标量值或者分数值。
8. <optgroup>	在一个web表单中, HTML元素 <optgroup> 会创建包含在一个 <select> 元素中的一组选项
9. <option>	在web表单中,  HTML元素 <option>  用于定义在<select>,  <optgroup> 或<datalist> 元素中包含的项。<option> 可以在弹出窗口和 html 文档中的其他项目列表中表示菜单项。
10. <output>	HTML <output> 标签表示计算或用户操作的结果。
11. <progress>	HTML中的progress (<progress>) 元素用来显示一项任务的完成进度.虽然规范中没有规定该元素具体如何显示,浏览器开发商可以自己决定,但通常情况下,该元素都显示为一个进度条形式.
12. <select>	HTML select (<select>) 元素是一种表单控件，可创建选项菜单。菜单内的选项为<option> , 可以由 <optgroup> 元素分组。选项可以被用户预先选择。
13. <textarea>	HTML <textarea> 元素表示一个多行纯文本编辑控件。
```
